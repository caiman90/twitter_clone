Getting Started
These instructions will get you a copy of the project up and running on your local machine for development and testing purposes.

Prerequisites
Following things needs to be installed on your local machine so you could run the web application

Node.js   -  [https://nodejs.org/en/]
Application information
Following technologies are used in web application:

React
Javascript
Bootstrap
NodeJS
MongoDB
Installing
Simply clone this repo using

git clone https://github.com/caiman90/twitter_clone.git

then install the Node modules with
npm install

then just start client with
webpack-dev-server

then navigate to server folder and start server with
node server.js
